# ERP AI RAG System

AI-powered backend system that allows natural language queries over company databases.

## Features
- Natural language → SQL generation
- RAG architecture
- Multi-company database support
- REST API
- LLM integration
- Schema-aware querying
- Fine-tuning ready structure

## Tech Stack
- Python
- Flask
- MSSQL
- LLM API
- RAG architecture
- HuggingFace
- SQL generation

## Project Structure

config/ → configuration files  
rag/ → RAG pipeline  
sql_ai/ → NL to SQL system  
api/ → backend API  
web/ → web interface  
schema/ → DB schema tools  
finetuning/ → training modules
